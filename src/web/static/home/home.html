About the Project

This project analyzes stablecoin transaction flows on EVM-compatible networks. A unified ERC-20 event extraction pipeline collects transfer data across Ethereum, Polygon, Arbitrum, Optimism, and Avalanche. Transaction graphs are constructed per network and stablecoin, and PageRank is applied to identify structurally influential addresses. Wallets are classified into entity types — CEX, Bridge, x402 agents, and regular wallets — to enrich the analysis. The dashboard provides three interactive tools for exploring the dataset: a graph-based PageRank view, a per-wallet flow explorer, and an ecosystem overview that quantifies the share of volume and transactions attributable to x402 AI agents versus traditional entity types.
